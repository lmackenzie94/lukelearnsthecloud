Luke Learns The Cloud ☁️

  • Namecheap domain pointing to AWS Nameservers
  • static website hosted on an S3 bucket
    • two buckets: one to host the website (`lukelearnsthe.cloud`) and another one to redirect the subdomain (`www.lukelearnsthe.cloud`) to the root bucket.
  • SSL Cert via Certification Manager
  • Two CloudFront distributions (one for `lukelearnsthe.cloud`; one for `www.lukelearnsthe.cloud`)
  • Route 53 to route traffic to the CloudFront distributions
  • Restricted S3 bucket access to CloudFront distribution via Origin Access Control (OAC).
    • With this, you can turn on “Block all public access” to make the S3 bucket reachable only through CloudFront.
